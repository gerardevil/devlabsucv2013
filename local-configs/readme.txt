El siguiente directorio contiene :

1.- settings.py - archivo de configuraciones para correr el proyecto con servidor local. manage.py runserver.

2.- view-example.html - una vista de ejemplo mostrando como deber ser ahora el acceso a datos locales, desde el proyecto como localhost.

3.- Crear un directorio static en la carpeta views y colocar el directorio assets en dicha carpeta.

[ WARNING ]
Cabe destacar que al momento de realizar la git push desde su repositorio local, debe primero verificar que las configuraciones setting.py sean las establecidas para el servidor webfaction, al igual que el metodo de acceso a los recursos estaticos.
